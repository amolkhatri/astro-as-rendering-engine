## Built-in Optimizations

| Feature | What it does |
|---------|-------------|
| `astro:assets` | Automatic image optimization, WebP/AVIF conversion |
| CSS Bundling | Automatic minification and dead code elimination |
| Prefetching | Instant page loads with `<link rel="prefetch">` |
| View Transitions | Native page transitions without full reload |
| Partial Hydration | Only ship JS for interactive components |

## Image Optimization

```astro
---
import { Image } from 'astro:assets';
import heroImage from '../assets/hero.png';
---

<!-- Automatically optimized! -->
<Image 
  src={heroImage} 
  alt="Hero" 
  width={800} 
  format="webp"
  quality={80}
/>
```

## View Transitions

```astro
---
import { ViewTransitions } from 'astro:transitions';
---

<html>
  <head>
    <ViewTransitions />
  </head>
  <body>
    <!-- Smooth page transitions! -->
  </body>
</html>
```
