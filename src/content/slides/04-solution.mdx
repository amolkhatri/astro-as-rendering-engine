## What Changes

| Aspect | Old Way | New Way |
|--------|---------|---------|
| **Server** | Renders + Bundles JS | Renders HTML only |
| **Output** | HTML + 2-5MB JS | HTML + ~0KB JS |
| **CDN** | Caches, breaks features | Caches perfectly |
| **Browser** | Downloads, parses, hydrates | Just renders HTML |
| **Interactivity** | Full page hydration | Islands only |
| **TTI** | 4-12 seconds | Instant |
